# GameOn

Site web de réservation pour événements gaming, entièrement codé from scratch en HTML/CSS/JavaScript.

## Comment le démarrer

Ce projet est un site statique HTML/CSS/JavaScript. Pour le lancer, il est recommandé d'utiliser un service comme l'extension Live Server dans VSCode :

1. Installez l'extension Live Server dans VSCode si ce n'est pas déjà fait.
2. Ouvrez le dossier du projet dans VSCode.
3. Clic droit sur [`index.html`](index.html) et sélectionnez "Open with Live Server". Cela lancera un serveur local (généralement sur http://127.0.0.1:5500)
